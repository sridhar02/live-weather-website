## Run Locally

Clone the project

```bash
  git clone https://github.com/sridhar02/live-weather-website
```

Go to the project directory

```bash
  cd live-weather-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Deployment

To deploy this project run

```bash
  npm run build
```
